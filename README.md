## Freda 是一个基于JavaEE的博客系统
### 技术框架：JDK1.8, Spring4.0.2, Mybatis3.2.6, Bootstrap3
### 数据库：MySQL5.7
### 富文本编辑器：summernote 
#### 推荐使用Chrome插件马克飞象markdown语法写文章
