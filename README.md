## Freda 是基于Java的博客系统
### 技术框架：JDK1.8, Spring4.0.2, Mybatis3.2.6, Bootstrap3
### 数据库：MySQL5.7
### 富文本编辑器：summernote
### MarkDown编辑器：editor.md
